Términos del Servicio

1. Uso del Sistema: Esta plataforma es un proyecto académico de DevSecOps. El usuario es responsable de salvaguardar su Contraseña Maestra.

2. Criptografía: El sistema utiliza cifrado Zero-Knowledge. Los administradores del sistema no tienen la capacidad técnica para recuperar contraseñas perdidas ni leer la información almacenada.

3. Responsabilidad: El software se proporciona "tal cual". Se prohíbe el uso de esta plataforma para almacenar información relacionada con actividades ilícitas.

Aviso de Privacidad

Recopilación de Datos: Solo almacenamos el nombre de usuario (Alias) y el hash iterado de su contraseña maestra para fines de autenticación.

Cero Rastreo: No utilizamos cookies de rastreo de terceros, píxeles ni herramientas de telemetría invasiva. Las cookies de sesión son estrictamente necesarias para el funcionamiento del portal.

Auditoría: Se registra la dirección IP y los intentos de acceso en archivos locales (Logs) estrictamente para fines de defensa contra ataques de fuerza bruta, bloqueos temporales y forense digital.

Documentación API

Los endpoints públicos (REST API) se encuentran actualmente deshabilitados por protocolos de seguridad (Hardening).

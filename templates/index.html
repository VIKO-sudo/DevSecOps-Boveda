Especificaciones del Sistema

Autenticación Robusta

Integración Continua (CI) con análisis estático de seguridad (SAST). El despliegue en el servidor se bloquea automáticamente si se detectan vulnerabilidades en el código fuente.

Auditoría (Logs)

Registro inmutable de eventos. Cada acción (login, acceso, eliminación) genera una traza forense en el servidor para cumplimiento normativo y detección de intrusiones.

Ciclo DevSecOps

Integración Continua (CI) con análisis estático de seguridad (SAST) mediante Bandit. El despliegue se bloquea automáticamente si se detectan vulnerabilidades en el código.

Términos de Servicio y Privacidad

Términos de Servicio
1. Responsabilidad de Credenciales

El usuario es el único responsable de mantener la confidencialidad de su contraseña. El administrador del sistema no tiene acceso a las contraseñas originales debido al cifrado unidireccional.

2. Política de Logs

Al utilizar este sistema, usted acepta que sus acciones (incluyendo dirección IP y marca de tiempo) sean registradas con fines de auditoría de seguridad y forense digital.

3. Limitación de Responsabilidad

Este software se proporciona "tal cual", sin garantía de ningún tipo. No nos hacemos responsables por pérdida de datos derivada del mal uso de la plataforma o fallos de hardware local.
